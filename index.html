HTML & CSS Cheat Sheet

HTML Tags

Name Description Tag
Html Contains all HTML code <html>
Anchor element Creates a link to an internal/external page or content <a>
Table Container for a HTML table <table>
Section Defines elements like chapters <section>
Article Holds content that makes sense on it's own <article>
Aside Contains info related to the main context <aside>
Figure Container for all media <figure>
Figure caption Describes the media in <figure> <figCaption>

CSS Selectors

Selector Description
Visibility To hide/show elements
Position Specifies the position of the element
z-index Set how far back /forward an element appears on a page (useful for overlapping elements)
Display Controls how elements flow in a document
